## DDL-表操作-查询
### 查询当前数据库所有表
```sql
show tables;
```
### 查询表结构
```sql
desc 表名;
```
### 查询指定表的建表语句
```sql
show create table 表名;
```

## DDL-表操作-创建
```sql
create table 表名(
    字段1 字段1类型[coment 字段1注释],
    字段2 字段2类型[coment 字段1注释],
    字段3 字段3类型[coment 字段1注释],
    ......
    字段n 字段n类型[coment 字段1注释],
)[comment 表注释];
```
例如：
```sql
create table tb_user(
    id int comment '编号',
    name varchar(50) comment '姓名',
    age int comment '年龄',
    gender varchar(1) comment '性别'
) comment '用户表';
```
