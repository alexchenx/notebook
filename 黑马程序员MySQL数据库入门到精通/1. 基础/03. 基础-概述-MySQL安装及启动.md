## MySQL数据库版本
 - 社区版（MySQL Community Server）
   
    免费，MySQL不提供任何技术支持

 - 商业版（MySQL Enterprise Edition）

   收费，可以试用30天，官方提供技术支持

本课程采用MySQL社区版 MySQL Community Server 8.0.26

## 下载（Windows）
下载地址：https://downloads.mysql.com/archives/installer/
选择下载：mysql-installer-community-8.0.26.0.msi

## 安装（Windows）
1. 双击安装程序开始安装
2. Choosing a Setup Type 选择 Developer Default, Next
3. Check Requirements 点击 Execute 安装所需依赖，然后 Next
4. Installation 点击 Execute 开始安装，然后 Next
5. Product Configuration 选择 Next 进入MySQL Server 配置
   - Type and Networking，都不用动保持默认, Next
   - Authentication Method, 保持默认，Next
   - Accounts and Roles, 设置密码为 123456 ， Next
   - Windows Service ，这里可以看到 MySQL在Windows下设置的服务名为 MySQL80 ，保持默认，Next
   - Apply Configuration， 点击 Execute，然后 Finish
6. 重新回到 Product Configuration 选择 Next 进入 MySQL Router配置
   - MySQL Router Configuration，点击 Finish
7. 重新回到 Product Configuration 选择 Next 进入 Samples and Examples配置
   - Connect To Server， password 填入之前设置的密码 123456 点击 Check, 然后 Next
   - Apply Configuration， 点击 Execute，然后点击 Finish
8. 重新回到 Product Configuration 选择 Next
9. Installation Complete，点击 Finish

## 服务启动与停止
方式一：在系统的服务中找到 MySQL80 服务手动进行启动和停止

方式二：使用命令行操作，使用管理员身份打开 cmd
```bash
# 启动
net start mysql80

## 停止
net stop mysql80
```

## 客户端连接
### 方式一：
自带程序 `MySQL 8.0 Command Line Client`

### 方式二：CMD命令行
1. 要使用CMD命令行需要先配置环境变量，将 C:\Program Files\MySQL\MySQL Server 8.0\bin 添加 到 Path
2. 打开cmd，按照下面格式进行连接
   ```bash
   mysql [-h 127.0.0.1] [-P 3306] -u root -p
   ```
